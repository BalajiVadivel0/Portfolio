# Assets Folder

This folder contains downloadable files and media for your portfolio.

## Files

### resume.pdf
- Replace this placeholder PDF with your actual resume
- Make sure the filename stays as `resume.pdf` or update the link in `index.html`
- Recommended: Keep file size under 2MB for faster downloads

## Adding More Files

You can add other downloadable files here:
- Certificates (PDF format)
- Portfolio pieces
- Case studies
- Any other documents you want visitors to download

## Image Assets

If you want to add images for your projects or profile:
1. Create an `images` subfolder
2. Add your images there
3. Update the image paths in `index.html`
